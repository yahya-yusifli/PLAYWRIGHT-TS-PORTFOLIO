A complete end-to-end test automation framework designed to demonstrate **industry-level best practices** for modern e-commerce testing.

This project focuses on **clean architecture, maintainability, and real-world scalability**, rather than simple demo tests.

### key highlights

- **modern architecture**  
  Page Object Model (POM) implemented with TypeScript and strict typing

- **comprehensive test coverage**  
  50+ test cases covering critical user journeys such as login, cart, checkout, and validation flows

- **ci/cd integration**  
  Fully automated execution using GitHub Actions with parallel test runs

- **professional reporting**  
  HTML and Allure reports with screenshots, videos, and trace support

- **performance optimized**  
  Parallel execution, test sharding, and reusable authentication fixtures
